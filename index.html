OSIRIS AquiLLM Capstone 2026

A Tool for Preserving Knowledge in Research Groups

Our Team: Bernie Boscoe, Chandler Campbell, in conjunction with the UCLA Astrophysics Datalab. They have built a promising AI tool called AquiLLM, and they are excited to see us further develop and improve it.

What is AquiLLM?

AquiLLM is an AI chatbot built specifically for research groups to preserve and access knowledge. It allows users to upload various document types from research papers to meeting notes and query them using natural language. With robust privacy options supporting both cloud-based and offline local models, AquiLLM helps research teams maintain institutional knowledge and onboard new members more effectively.

This is a page for the Capstone 2026 OSIRIS AquiLLM Project. For more information on the broader AquiLLM Project, please see AquiLLM's official website here: https://aquillm.org/

Where We Come In

AquiLLM is a work in progress and is often used with local models which can be prone to hallucinations. Therefore, being able to rate AquiLLM's responses is an important step in understanding which situations the tool falters and what areas could use fine tuning.

The Current Challenge

There is an existing rating system in place where users can give numerical ratings from 1-5 on each of AquiLLM's responses, however the system is incomplete. The ratings are not stored anywhere outside the conversation, and there is no other feedback system aside from the basic 1-5 rating scale (no functionality to allow the users to explain which part of the response was wrong).

Our Mission

AquiLLM developers are interested in being able to critically evaluate the performance of AquiLLM so that they can figure out how to improve it. In order to evaluate the responses however, the existing rating system needs to be completed and expanded upon to give developers more information to work with. This is step 1 in improving AquiLLM, and later steps in the coming months will include building a system to seamlessly integrate data from the astronomical instrument OSIRIS (OH-Suppressing Infrared Imaging Spectrograph), based in Hawaii.
